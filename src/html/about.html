About Me

I'm Hichem Gouia, a web developer and designer based in France, moving between Paris and Nancy.

I create elegant and minimalist designs inspired by Awwwards-winning websites. My style is characterized by sleek lines, tasteful fonts, and sophisticated visuals that embody clarity and sophistication.

I am proficient in various programming languages and tools that enable me to bring unique and engaging designs to life, using Figma to create compelling visual identities and wireframes.

I have technical expertise in Node.js, Express.js, TypeScript, Sequelize, EJS, Passport.js, and MongoDB. I excel in crafting seamless user interfaces with VueJS 3 and Ionic, using CSS libraries like TailwindCSS and Bootstrap for a polished finish, and incorporating Locomotive Scroll for added fluidity and dynamism.
